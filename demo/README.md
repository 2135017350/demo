✅ 三、交易系统（核心）
1. 一口价购买

下单 → 支付 → 商家发货 → 确认收货 → 完成

2. 集换（Swap）功能

二次元卡片 / 周边交易的核心功能

用户 A 列出愿出商品（Offer）

用户 B 提供等价商品（Want）

双方同意 → 平台撮合 → 各自寄件 → 完成交换
或者

平台托管物品（寄存换）

3. 出价 / 议价系统

买家出价

卖家接受 / 拒绝 / 还价

4. 托管支付（推荐）

资金托管至平台

收货确认后放款，减少诈骗风险

5. 订单管理

买家订单、卖家订单列表

发货信息填写

快递单号跟踪
